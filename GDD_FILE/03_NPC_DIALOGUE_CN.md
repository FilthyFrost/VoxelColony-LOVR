# 03 - NPC中文对话系统

## 设计意图

NPC说话是游戏个性的核心。语气定位：**欠揍的小暴君** — 像一个被宠坏的小孩+黑心甲方+嘴硬傲娇的混合体。关键：搞笑，不是真的恶毒。玩家应该笑着骂"这小子真欠揍"，而不是真的生气。

## 语气规则

1. 短句为主（6-15个字），气泡空间有限
2. 大量使用感叹号和省略号
3. 偶尔用网络用语增加亲切感
4. 极少夸奖，夸也是嘴硬式的（"没想到你还有点用"）
5. 永远不说"谢谢"（说了就不欠揍了）
6. 可以第四面墙（"你是不是挂机了？"）

## 对话显示机制

### 气泡格式

```
┌──────────────────────────┐
│  喂！给我10个圆石！快点！    │  ← 骂人台词（每5-8秒随机切换）
│  ████████░░  圆石 8/10     │  ← 进度条 + 材料计数器（实时更新）
└──────────────────────────┘
         ▽
       [NPC]
```

### 气泡切换规则

1. `first` 台词：需求刚产生时显示一次（3秒）
2. `waiting` 台词：之后每 5-8 秒随机选一句（随机间隔防止机械感）
3. `progress` 台词：玩家每投放一个正确材料时立即显示（2秒），然后回到 `waiting` 循环
4. `fulfilled` 台词：需求满足时显示一次（3秒），然后切换到 `BUILDING` 台词
5. 紧急度影响切换速度：`critical` 时每 3 秒换一句（更急促）

## 性格台词后缀

NPC的性格特质会在通用台词后追加一个短后缀，用最低成本让每个NPC听起来不一样。

    lazy = {
        "...好累啊...",
        "...能不能偷懒...",
        "...不想动...",
        "...算了算了...",
    },
    greedy = {
        "...多来点！",
        "...不够不够！",
        "...还要更多！",
        "...翻倍！",
    },
    shy = {
        "...拜托了...",
        "...那个...",
        "...小声说...",
        "...可以吗？",

### 使用方式

```lua
function Dialogue.getLine(category, subcategory, npc, vars)
    -- 1. 从对话池中随机选一句基础台词
    local pool = DIALOGUE[category][subcategory]
    local base = pool[math.random(#pool)]
    
    -- 2. 替换模板变量
    base = base:gsub("{count}", vars.count or "?")
    base = base:gsub("{material}", vars.material or "?")
    base = base:gsub("{delivered}", vars.delivered or "?")
    base = base:gsub("{remaining}", vars.remaining or "?")
    
    -- 3. 追加性格后缀（50%概率，避免每句都带后缀）
    if math.random() < 0.5 then
        for trait, _ in pairs(npc.traits) do
            local suffixes = PERSONALITY_SUFFIX[trait]
            if suffixes then
                base = base .. suffixes[math.random(#suffixes)]
                break  -- 只追加一个特质的后缀
            end
        end
    end
    
    return base
end
```

效果示例：
- diligent NPC: "给我10个圆石！快点！效率效率！"
- lazy NPC: "给我10个圆石！快点！...好累啊..."
- shy NPC: "给我10个圆石！快点！...拜托了..."
- greedy NPC: "给我10个圆石！快点！...多来点！"
